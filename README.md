# Huml

Huml is a template engine derived from [Haml](github.com/haml/haml).
Like the relationship between Sass and Scss, Huml uses braces for nesting structure
when Haml uses indentations.

## Formatting

    doctype html
    %html {
      %head {
      }

      %body {
      }
    }
